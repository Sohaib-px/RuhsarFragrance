# Ruhsar - The Fragrance Of The Soul

A modern, responsive website for Ruhsar fragrance brand built with Next.js, React, TypeScript, and Tailwind CSS.

## Features

- 🎨 Modern and elegant design with golden theme
- 📱 Fully responsive for all devices
- ⚡ Fast performance with Next.js
- 🎯 Multiple sections: Home, About, Contact, FAQ, Reviews, Article
- 🔗 Social media integration (Instagram, WhatsApp, Facebook)
- 📧 Contact form for customer inquiries
- 📝 Interactive FAQ section
- ⭐ Customer reviews showcase

## Pages & Sections

- **Home**: Hero section with call-to-action
- **About**: Brand story and mission
- **Contact**: Contact form for inquiries
- **FAQ**: Frequently asked questions with accordion
- **Reviews**: Customer testimonials with ratings
- **Article**: Blog-style article about perfumery

## Getting Started

### Prerequisites
- Node.js 16+ and npm

### Installation

1. Install dependencies:
```bash
npm install
```

### Development

Run the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the website.

### Build

Build for production:
```bash
npm run build
```

### Start Production Server

```bash
npm start
```

## Technology Stack

- **Framework**: Next.js 15
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **UI Components**: React

## Customization

- Update logo: Replace in `Header.tsx`
- Modify colors: Edit `tailwind.config.ts` and `globals.css`
- Add social media links: Update `Footer.tsx`
- Add/edit FAQ items: Modify `components/FAQ.tsx`
- Update reviews: Edit `components/Reviews.tsx`
